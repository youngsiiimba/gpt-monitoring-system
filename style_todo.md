# **POC Styling Theme Guide – Monitoring & Reporting System**

## **1. Color Palette (Neutral Professional)**

| Role                 | Color                            | Usage                             |
| -------------------- | -------------------------------- | --------------------------------- |
| Primary Background   | `#FFFFFF` (white)                | Main page background              |
| Secondary Background | `#F4F4F4` (light gray)           | Cards, tables, secondary sections |
| Text Primary         | `#1F2937` (dark gray/near black) | Headings, body text               |
| Text Secondary       | `#4B5563` (medium gray)          | Subtext, labels, descriptions     |
| Accent / Highlight   | `#D63636` (muted red, GPT brand) | KPI status, important alerts      |
| Borders / Dividers   | `#E5E7EB` (light gray)           | Tables, cards, sections           |
| Chart Colors         | `#1F2937`, `#4B5563`, `#D63636`  | Lines, bars, or pie segments      |

**Rules:**

* No flashy colors.
* Only **one accent color** allowed for emphasis.
* Maintain contrast for readability.

---

## **2. Typography**

* **Font Family:** `Inter, sans-serif` (professional, modern)
* **Heading Levels:**

  * H1: 28px / Bold / #1F2937
  * H2: 24px / Semi-Bold / #1F2937
  * H3: 20px / Medium / #1F2937
* **Body Text:** 16px / Regular / #1F2937
* **Secondary Text / Labels:** 14px / Medium / #4B5563

**Rules:**

* Consistent font usage across all pages.
* Avoid decorative fonts.
* Line height: 1.5 for readability.

---

## **3. Spacing & Layout**

* **Grid System:** 8px increments
* **Page Margins:** 24px (desktop), 16px (mobile)
* **Card Padding:** 16px
* **Table Padding:** 12px
* **Section Spacing:** 32px between major sections

**Rules:**

* Uniform spacing for a clean, structured look
* Avoid clutter or overlapping elements

---

## **4. Components Styling**

### **4.1 Cards / KPI Widgets**

* Background: #F4F4F4
* Border: 1px solid #E5E7EB
* No shadow or glow
* Rounded corners: 4px
* Content: minimal text, optional small icon on top left
* KPI color coded with accent color when below/above thresholds

### **4.2 Tables**

* Minimal borders only for separation
* No row stripes
* Header bold (#1F2937), body (#4B5563)
* Hover: subtle background #E5E7EB for row highlight
* Columns: aligned, no text wrapping unless necessary

### **4.3 Charts**

* Simple line/bar/pie charts
* Colors limited to palette
* No animations, no gradients
* Labels clear and minimal

### **4.4 Buttons**

* Flat, solid fill: #1F2937
* Hover: lighten background slightly (#4B5563)
* Text: white (#FFFFFF)
* No shadows, no animations except subtle hover

### **4.5 Filters / Inputs**

* Border: #E5E7EB, rounded 4px
* Hover/focus: subtle border color change #4B5563
* Labels: #4B5563
* Placeholder: #9CA3AF (light gray)

---

## **5. Icons & Imagery**

* Only minimal monochrome icons
* Color: #4B5563 or #1F2937
* Purpose: clarify actions, not decorative
* Size: 16–20px

---

## **6. Interactivity**

* Hover: subtle background or border change
* Click: subtle opacity/scale effect
* No flashy transitions, no animations that distract
* All buttons, filters, links must have clear visual feedback

---

## **7. Dark Mode**

* **Background:** #1F2937 (dark gray)
* **Secondary:** #111827
* **Text Primary:** #F9FAFB (white)
* **Text Secondary:** #D1D5DB (light gray)
* **Accent:** #D63636 (same muted red)
* **Cards:** #111827
* **Tables:** subtle borders #374151
* **Charts:** same palette, colors adjusted for contrast

**Rules:**

* Maintain contrast ratios for accessibility
* All components follow same layout & spacing as light mode
* Dark mode toggle optional for POC but desirable

---

## **8. General Rules / Do’s and Don’ts**

**Do:**

* Keep design consistent across pages
* Use accent color sparingly
* Keep charts and tables simple and readable
* Maintain spacing and alignment rules strictly

**Don’t:**

* Use gradients, shadows, glows, or flashy effects
* Overload pages with information
* Use multiple font families
* Use bright or clashing colors
